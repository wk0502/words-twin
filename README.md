# WordsTwin

WordsTwin 是一个 Windows 桌面端易混词对比学习工具。它会把相似单词成对展示在桌面弹窗中，支持点击切换、自动切换，以及 IELTS / CET-6 / 日常用语词库下载更新。

- [单独下载安装器 WordsTwin_Setup.exe](https://github.com/wk0502/words-twin/releases/download/v1.0.0/WordsTwin_Setup.exe)

## 适合谁

- 正在备考 IELTS / CET-6 的英语学习者
- 容易混淆形近词、义近词的用户
- 写英文内容、技术文档时想减少用词错误的人

## 核心功能

- 成对展示易混词，例如 `inspect / suspect`
- 支持点击切换下一组单词
- 支持按时间间隔自动切换
- 支持调整弹窗大小、透明度和自动显示
- 支持词库管理和云端词库下载更新

## 安装说明

1. 下载 `wordstwin.zip`。
2. 解压 ZIP。
3. 双击解压后的 `WordsTwin_Setup.exe`。
4. 如果 Windows SmartScreen 提示未知发布者，选择“更多信息”后继续运行。
5. 启动后可在系统托盘中打开设置、暂停/恢复、显示弹窗或退出。

高级用户也可以单独下载 `WordsTwin_Setup.exe` 和 `WordsTwin-1.0.0-win.zip`，但两个文件必须放在同一个文件夹。
